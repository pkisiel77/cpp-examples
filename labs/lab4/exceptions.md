# Wyjątki (Exceptions)

```cpp
try 
{
    // "Wyrzucenie" wyjątku:
    if(coś poszło nie tak)
        throw wyjątek;
    // KOD
}
catch (typ nazwa) 
{
    // Obsługa wyjątku
}
```
