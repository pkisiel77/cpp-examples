## Tablice jednowymiarowe

Tablica jest strukturą danych, która umożliwia przechowywanie w sposób zorganizowany wielu zmiennych tego sameog typu.

```cpp
// deklaracja
int array[2];
// definicja
array[0] = -1;
array[1] = 0;
array[2] = 20;
```
