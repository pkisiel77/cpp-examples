# Testy i scenariusze sprawdzenia

W tym katalogu zapisz testy automatyczne albo ręczne scenariusze sprawdzenia.

## Scenariusze ręczne

1. Uruchom program bez danych i sprawdź komunikat startowy.
2. Podaj poprawne dane i sprawdź wynik.
3. Podaj błędne dane i sprawdź, czy program nie kończy się awaryjnie.

## Testy automatyczne

Jeśli projekt ma testy automatyczne, dopisz tutaj:

- jak je skompilować,
- jak je uruchomić,
- jaki wynik oznacza sukces.
