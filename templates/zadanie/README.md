# Szablon krótkiego zadania

## Temat

Wpisz krótki tytuł zadania.

## Cel

Opisz jednym lub dwoma zdaniami, co program ma ćwiczyć.

## Kompilacja

```sh
c++ -std=c++17 -Wall -Wextra -pedantic main.cpp -o zadanie
```

## Uruchomienie

```sh
./zadanie
```

## Scenariusze sprawdzenia

Przed oddaniem sprawdź:

- dane poprawne,
- dane graniczne,
- dane błędne,
- czy komunikaty dla użytkownika są czytelne.

## Notatki

Wpisz krótko, co było najtrudniejsze albo co wymaga dalszej poprawy.
