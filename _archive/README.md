# Archiwum

Luźne linki, starsze notatki i materiały, które nie są częścią głównej ścieżki kursu.

## Zawartość

- [old-labs](old-labs) - starsze linki i materiały laboratoryjne.
- [old-tnt](old-tnt) - wcześniejsze notatki, zadania i materiały powiązane z projektami TNT.

## Zasada pracy

Materiały z tego katalogu można później przenieść do segmentów, jeśli zostaną rozbudowane do formatu lekcji: cel, wymagania wstępne, teoria, przykład, zadania i kryteria zaliczenia.
