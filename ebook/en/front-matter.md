# Introduction

- Author: Paweł Kisielewicz `<pkisiel@gmail.com>`
- Place and date: Krakow, 18 June 2026
- Version: `0.4.0-en`

This ebook is the first English draft based on the Polish C++ learning
materials. The Polish version remains the source of truth for now. The English
version starts with the course structure, learning path, and translation plan.

The full course is organized from environment setup and C++ basics to functions,
arrays, strings, files, exceptions, object-oriented programming, STL, build
systems, testing, and final projects.
