# Introduction

- Author: Paweł Kisielewicz `<pkisiel@gmail.com>`
- Place and date: Krakow, 20 June 2026
- Version: `0.7.1-en`

This ebook is the first complete English edition based on the Polish C++
learning materials. The Polish version remains the source of truth for now,
while the English version covers the full learning path from setup to final
projects.

The full course is organized from environment setup and C++ basics to functions,
arrays, strings, files, exceptions, object-oriented programming, STL, build
systems, testing, and final projects.
